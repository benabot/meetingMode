# Roadmap

## Phase 0

- Keep the project compiling and structurally clean.
- Lock the menu bar shell and settings entry point.

## Phase 1

- Add local preset persistence.
- Build preset CRUD.
- Introduce a simple active session lifecycle.

## Phase 2

- Implement permission-aware app launch and hide flows.
- Add a practical restore model for only what Meeting Mode changed.
- Add a lightweight overlay window.

## Phase 3

- Add onboarding only if permissions and first-run guidance justify it.
- Add polish around session status and preset ergonomics.
